## Cross-Codebase Patterns Discovered

- Entities use inner enums for status (`Order.OrderStatus`, `PurchaseOrder.PurchaseOrderStatus`)
- Constructor injection exclusively; no field `@Autowired`
- `LocalDate` used throughout (not `LocalDateTime`)
- `GlobalExceptionHandler` centralized exception mapping
- No DTOs yet; entities returned directly from controllers
- Seed data in `data.sql` — plain INSERT statements grouped by entity type
- `@GeneratedValue(strategy = GenerationType.IDENTITY)` for all primary keys
